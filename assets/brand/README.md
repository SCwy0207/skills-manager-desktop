# Skills Manager brand assets

These identity assets are covered by the repository's
[`BRAND_LICENSE.md`](../../BRAND_LICENSE.md), not the MIT software licence.

The production mark is the original **Skill Relay** symbol: one local management
hub fans out to three Agent targets inside a repository frame.
It is designed to stay readable at Windows taskbar and installer sizes.

- Primary ink: `#0B1324`
- Product indigo: `#5368ED`
- Local teal: `#16B7B1`
- Signal teal: `#59E3CF`
- Cloud surface: `#F5F8FD`

Use `skills-manager-mark.svg` as the vector source. Raster app icons and the
native NSIS/WiX artwork are generated by `scripts/generate_brand_assets.py`.
Install the pinned generator dependency with
`python -m pip install -r scripts/requirements-brand.txt`.
The production multi-resolution ICO/ICNS set is generated with
`pnpm tauri icon assets/brand/skills-manager-app-icon.png --output src-tauri/icons`;
`skills-manager-app-icon.ico` is a convenience copy of that Tauri output.
The reference image supplied during design was used only for high-level colour
and geometric direction; no reference pixels, watermark, wordmark, or path
geometry are included here.
